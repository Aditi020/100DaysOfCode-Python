# Coffee Machine

This project simulates a coffee machine, allowing users to choose their coffee preferences and get a customized coffee.

## Features

- Choose from various types of coffee (e.g., espresso, latte, cappuccino).
- Customize the coffee with options like milk, sugar, etc.
- View the available ingredients and their quantities.
- Maintain an account of resources and deduct them based on the coffee made.

## Usage

1. Clone the repository.
2. Run the `coffee_machine.py` file to start the coffee machine simulation.
3. Follow the on-screen instructions to select and customize your coffee.

## Technologies Used

- Python
